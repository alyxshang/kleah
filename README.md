# KLEAH :owl: :feather:

***A federated blogging platform for the wise. :owl: :feather:***

## ABOUT :books:

This repository contains the source code for a federated social platform that uses ActivityPub for federating users and their activities. 
Kleah aims to compete and work with platforms like Mastodon, Misskey, and Sharkey, thus expanding the choice of ActivityPub servers for users.
This platform was conceptualized, designed, and implemented by me. Kleah also attempts to be a very stable and reliable Fediverse platform. 
***Kleah*** is named after an ancient sorceress famous for spreading wisdom and being a mystic.
