# TERMINOLOGY :book:

## Users

The admin user is called the [***Tianquan (天权)***](https://genshin-impact.fandom.com/wiki/Ningguang), "the (guiding) star (of wisdom)" of the instance.

Since ***Kleah*** is a platform for magic-wielding (ancient) creatures, a normal user is called a ***Mage***.

## User posts

Since everyone who posts on this platform is a magic-wielding user, any posts they make are called ***Charms***. You cast a ***Charm*** into the ether.

If one promotes or boosts a charm made by another user, this is not called "boosting". This is too basic a term for something as important as promoting
ancient, important knowledge. Promoting a charm is called ***proclaiming a charm***.
